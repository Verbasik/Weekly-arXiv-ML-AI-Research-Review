# Google Agent Development Kit Demo: Автоматический Git Workflow

[![Google ADK](https://img.shields.io/badge/Собрано_на-Google_ADK-blue)](https://github.com/google/adk-docs)
[![Python](https://img.shields.io/badge/Язык-Python-green)](https://python.org)
[![MCP](https://img.shields.io/badge/Интеграция-MCP-orange)](https://modelcontextprotocol.io)

Этот репозиторий содержит демонстрацию автоматического Git workflow с использованием [Google Agent Development Kit (ADK)](https://github.com/google/adk-docs). Проект демонстрирует, как ИИ-агенты могут автоматически анализировать изменения в коде, создавать подробные коммиты и управлять Git репозиторием.

![Скриншот демонстрации](assets/screenshot.jpg)

## 🎯 Функциональность

### Основные возможности
- **Автоматический анализ изменений**: Агент сканирует указанную директорию и анализирует все изменения
- **Интеллектуальные коммиты**: Создание подробных и информативных сообщений коммитов
- **Git интеграция**: Автоматическое добавление файлов в индекс, коммит и пуш
- **Веб-интерфейс**: Современный UI для взаимодействия с агентом
- **MCP интеграция**: Использование Model Context Protocol для работы с файловой системой

### Сценарии использования
1. **Автоматизация разработки**: Агент анализирует изменения после работы над кодом
2. **Code Review**: Интеллектуальный анализ изменений с подробным описанием
3. **Документирование**: Автоматическое создание описаний изменений
4. **CI/CD интеграция**: Возможность интеграции в процессы непрерывной разработки

## 🚀 Быстрый старт

### Предварительные требования
- Python 3.9+
- Node.js 18+
- Git
- Google AI API ключ

### Установка

1. **Клонирование репозитория**
```bash
git clone <repository-url>
cd Google-Agent-Development-Kit
```

2. **Настройка Python окружения**
```bash
python -m venv .venv
source .venv/bin/activate  # macOS/Linux
# или
.venv\Scripts\activate     # Windows
```

3. **Установка зависимостей**
```bash
pip install -r requirements.txt
cd ui && npm install
```

4. **Настройка API ключа**
```bash
export GOOGLE_API_KEY=your_google_ai_api_key
```

### Запуск

1. **Запуск ADK агента**
```bash
cd src
adk api_server
```

2. **Запуск веб-интерфейса**
```bash
cd ui
npm run dev
```

3. **Открытие в браузере**
```
http://localhost:3000
```

- **Настройки агента**: Конфигурация поведения и стиля коммитов

## 🔧 Конфигурация

### Переменные окружения
```bash
GOOGLE_API_KEY=your_google_ai_api_key
GOOGLE_GENAI_USE_VERTEXAI=FALSE
ADK_AGENT_MODEL=gemini-2.0-flash
ADK_AGENT_NAME=git_workflow_agent
```

### Настройки агента
```python
# src/git_workflow_agent/config.py
AGENT_CONFIG = {
    "model": "gemini-2.0-flash",
    "name": "git_workflow_agent",
    "description": "Автоматический анализ изменений и создание коммитов",
    "instruction": """
    Ты - эксперт по анализу изменений в коде. Твоя задача:
    1. Анализировать изменения в файлах
    2. Создавать информативные сообщения коммитов
    3. Следовать лучшим практикам Git
    """
}
```
